# Air Draw

A browser-based hand gesture drawing app using MediaPipe Hands.

## Gestures

- Idle — No drawing
- 1 Finger — Draw
- 2 Fingers — Highlight
- Palm — Erase

## Features

- Draw in the air
- Highlighter mode
- Eraser mode
- Colour picker
- Undo / Redo
- Clear canvas
- Webcam hand tracking

## Run

Open `index.html` in a modern Chromium-based browser and allow camera access.

## Technologies

- HTML
- CSS
- JavaScript
- MediaPipe Hands (CDN)
